== Description ==

MotoPress Content Editor enhances the standard WordPress editor and enables to build websites visually. It's complete solution for building responsive pages without coding and simply by dragging and dropping content elements. MotoPress plugin is easily extendable, fully responsive and works with all WordPress themes.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
You'll find 'MotoPress' in your Wordpress admin panel.

Read the [usage documentation](http://www.getmotopress.com/documentation/wordpress/plugins/) on the MotoPress website
